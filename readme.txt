Instrucciones para ejecutar el programa ejecute el programa desde el directorio tarea3/target con el comando:

java -cp tarea3-CSOF5101-1.0-SNAPSHOT.jar edu.uniandes.ecos.tarea3.app.App c:\testFile.txt 386

donde "c:\testFile.txt" corresponde al archivo a cargar y "386" al tamaño del proxy

Se puede verificar el funcionamiento en heroku en la siguiente url:

https://morning-taiga-73451.herokuapp.com/regresionLineal?proxy=386  

puede cambiar el valor de proxy=386 para obtener resultado diferente. se adjuntan las fuentes que se cargan en heroku en el archivo tarea3Heroku.zip dentro de la carpeta src.
